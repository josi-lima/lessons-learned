"Just the thought of learning to code can be very intimidating. It implies a technical form of communication that computers, not humans, are meant to understand. One way many people start learning to code is by picking a popular programming language and jumping in head first with no direction. This could take the form of an online coding course, a tutorial project, or a random book purchase on a specific topic. Rarely do prospective developers start with a roadmap, which in my opinion is the ideal way to go!"
